GitHub-first

  • GitHub הוא מקור האמת היחיד – מצב הפרויקט נקבע לפי מה שבמאגר.
  • לא מסתמכים על מצב מקומי – קבצים/שינויים שלא נמצאים ב-commit לא נחשבים קיימים.
  • כל שינוי משמעותי חייב לעבור דרך git commit ואז git push.
