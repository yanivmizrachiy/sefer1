# אתר — מידע

האתר של יניב רז.

קישור: `https://yanivmizrachiy.github.io/sefer1/`

## קבצים

- `index.html` – העמוד הראשי
- `hishtalmut.html` – עמוד ההשתלמויות
- `hishtalmut-data.json` – נתוני ההשתלמויות (נטענים ע"י `hishtalmut.html`)
- `styles.css` – עיצוב

## איך מריצים מקומית

מומלץ להריץ דרך VS Code כדי לקבל תצוגה חיה שמתעדכנת מייד בתוך חלון פנימי (Simple Browser):

- Terminal → Run Task… → `DEV: Persistent Mode (serve + external browser + VS Code Simple Browser)`

אפשר גם לפתוח את `אתר/index.html` בדפדפן, אבל זה לא כולל תצוגה חיה קבועה.
